# Coursera: Perform Exploratory Data Analysis on Retail Data with Python
Mini project from Coursera for conducting an exploratory data analysis using Python and Jupyter Notebook.

This project tasked me to explore and analyze a dataset from an online retail company to gain insights into the store's sales trends, customer behavior, and popular products. Ultimately, training me to serve as a data analyst to help interpret real-world data with the intention to make key business decisions for the future of the company.

# Objectives Achieved
- Learned how to load, clean, process, analyze, and visualize data using Python
- Explored the basic statistics of the dataset
- Identified outliers or anomalies in the dataset
- Produced rational conclusions with facts and summarized the findings resulting from the exploratore data analysis

# Skills Applied
- Python
- Numpy
- Matplotlib
- Git
- Jupyter Notebook
